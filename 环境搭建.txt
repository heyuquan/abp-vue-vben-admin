docker-compose 中的网络  （这样可以不同的compose使用同一个网络，不然Compose会为每一个app设置一个default网络）
https://blog.csdn.net/lclfans1983/article/details/113179582
第一步：
docker network create env-net
docker network create services-net
第二步：
networks:
  env-net:
    external: true    # 使用已存在的网络
  default:     # 设置默认使用的网络，如果没有在单个节点上设置 networks 时使用
    external:
      name: services-net


#、运行环境
本地运行环境约定，ip地址全部指向dev环境。如开发人员要调试本地修改的代码，则修改对应微服务的ip指向本地
dev环境：有相关环境基础设置，eg：数据库，redis，rabbitmq等等。和微服务
